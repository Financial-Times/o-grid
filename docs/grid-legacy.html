Using nesting to create more complex layouts

Divs can be broken down into a further 12 columns, by nesting another grid-row containing more grid-cols

0: data-o-grid-colspan="8"
0: data-o-grid-colspan="4"
1: data-o-grid-colspan="4"
2: data-o-grid-colspan="4"
1: data-o-grid-colspan="4"
0: data-o-grid-colspan="6"
1: data-o-grid-colspan="6"

Variable column spans (with large default and conditionally hidden column)

These columns are set to default to 12 columns (full width), but span 4 at the large mode (with one hidden), and 3 at the extra large mode.

0: data-o-grid-colspan="12 l4 xl3"
1: data-o-grid-colspan="12 l4 xl3"
2: data-o-grid-colspan="12 l4 xl3"
3: data-o-grid-colspan="12 l0 xl3"

Variable column spans (with small default and conditionally hidden column)

These columns are set to default to 3 columns, but span 4 at the large mode (with one hidden), 6 at the and medium mode, and 12 at the small mode.

0: data-o-grid-colspan="3 l4 m6 s12"
1: data-o-grid-colspan="3 l4 m6 s12"
2: data-o-grid-colspan="3 l4 m6 s12"
3: data-o-grid-colspan="3 l0 m6 s12"

Columns hidden by default which appear at larger layouts

All but one of these columns are hidden on smaller screens. On larger screens they should all take up a third of the space available

0: data-o-grid-colspan="0 l4 xl4"
1: data-o-grid-colspan="hide l4 xl4"
2: data-o-grid-colspan="12 l4 xl4"

Usage on existing pages

The grid can be applied within components on a page which doesn't use o-grid for its global layout by wrapping in a container with class o-grid-box

0: data-o-grid-colspan="8 s12"
0: data-o-grid-colspan="4"
1: data-o-grid-colspan="4"
2: data-o-grid-colspan="4"
1: data-o-grid-colspan="4 s12"
