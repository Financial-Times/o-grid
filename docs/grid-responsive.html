The 12 column grid

The grid is made up of 12 virtual columns. Actual columns can span 1 to 12 virtual columns.

Using nesting to create more complex layouts

Divs can be broken down into a further 12 columns, by nesting another grid-row containing more grid-cols

0: .ft-grid-col-d8
0: .ft-grid-col-d4
1: .ft-grid-col-d4
2: .ft-grid-col-d4
1: .ft-grid-col-d4
0: .ft-grid-col-d6
1: .ft-grid-col-d6

Variable column spans

These columns are set to default to 12 columns (full width), but span 6 at the large mode, and 4 at the extra-large mode.

0: .ft-grid-col-d12-xl3-l6
1: .ft-grid-col-d12-xl3-l6
2: .ft-grid-col-d12-xl3-l6
3: .ft-grid-col-d12-xl3-l6

Usage on existing pages

The grid can be applied within components on a page which doesn't use grid-module for its global layout by wrapping in a container with class ft-grid-box

0: .ft-grid-col-d8-s12
0: .ft-grid-col-d4
1: .ft-grid-col-d4
2: .ft-grid-col-d4
1: .ft-grid-col-d4-s12
